# README (english)

Project is built with Rails API

Install all dependencies
```
bundle
```

Setup Database
```
bin/rake db:setup
bin/rake db:migrate
```

Run tests
```
bin/rake
```
The default locale is `pt`, but you can explicitly select one between `en` and `pt` by prefixing the routes:
```
/en/students
/pt/students
```
